### Sorted Set
&ensp;&ensp; *为什么 Sorted Set 既能支持高效的范围查询，同时还能以 O(1) 复杂度获取元素权重值？*

&ensp;&ensp;这其实和 Sorted Set 的底层设计实现相关。 Sorted Set 支持范围查询，这是因为它核心数据结构采用了跳表，而它又能以常数复杂度获取
元素权重，这是因为它同时采用了哈希表进行索引。
